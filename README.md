# ONT : Wallet


#### Introduction
> ONT桌面测试版。


#### Start
目前：直接改index.html，引入sdk调试。

后期：复制vue dist里文件到/src/wallet/下。


#### Run
```
npm install
```

#### Dev
```
npm start
```

#### Packager
```
npm run package:mac
npm run package:win
npm run package:linux
```
App输出目录：./dist/
